From Bulbapedia, the community-driven Pokémon encyclopedia.

Spite (Japanese: うらみ Spite) is a non-damaging Ghost-type move introduced in Generation II. It is TM177 in Generation IX.

Effect

Spite decreases the PP of the last move the target used. Spite fails if that move has no PP remaining.

Generation II

Spite decreases the move's PP by 2 to 5 (selected at random). If the amount that would be lost is greater than the amount remaining, the move loses all of its PP.

If the Pokémon's most recent move is Metronome, it's considered to be the move Metronome turned into; as Metronome is incapable of calling moves the user already knows in this generation, Spite will always fail against Metronome.

Generation III

Spite fails if that move has exactly 1 PP remaining.

If the Pokémon's most recent move is Metronome or another move that calls other moves, Spite will now correctly deduct PP from the base move instead of failing.

Spite can be used as the second move of a Pokémon Contest combination, with the user gaining 1 bonus appeal point if Curse was used in the previous turn.

Generation IV

Spite decreases the move's PP by exactly 4 (unless the move has fewer than 4 PP remaining, in which case the number left to deduct is correctly shown).

Spite no longer fails if the targeted move has exactly 1 PP remaining.

Generation V onwards

Spite can now be reflected with Magic Coat.

Spite can also be used as part of a Contest Spectacular combination, with the user gaining extra three appeal points if any of the moves Encore, Taunt or Torment was used in the prior turn.

If powered up by a Ghostium Z into Z-Spite, the user regains all of its HP.
